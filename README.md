Este portfólio foi feito com HTML, CSS e JavaScript para mostrar um pouco sobre mim, minhas habilidades e alguns projetos iniciais. O site tem um visual em tons de lavanda, funciona bem no computador e no celular, e inclui uma apresentação, uma parte sobre mim, meus projetos e um formulário de contato. O JavaScript cuida do menu e da validação do formulário. É um projeto simples para treinar e acompanhar minha evolução no desenvolvimento.
